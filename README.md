# SolidWorks Plugin for Logi Options

This plugin provides a set of commands for controlling SolidWorks using Logi Options. It allows you to perform various actions in SolidWorks directly from your Logitech device, enhancing your workflow and productivity.

## Features

- **Sketch Mode**: Toggle sketch mode in SolidWorks.
- **Extruded Boss/Base**: Create an extruded boss/base feature.
- **Corner Rectangle**: Create a corner rectangle sketch entity.
- **Add Relation**: Add a relation between sketch entities.
